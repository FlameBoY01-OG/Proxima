# ui/ (Phase 5)

React + Vite + TypeScript + shadcn/ui. The demo surface:

- **Vector-space map** — vectors projected to 2D (backend PCA/UMAP endpoint),
  rendered as a scatter; search highlights the query + draws lines to its kNN.
- **Search playground** — input → ranked results with scores.
- **Metadata filter panel** — genre/year/studio toggles re-run filtered search.
- **Live metrics** — latency, recall@10, QPS, vector count.
- **Data controls** — Seed demo data / Clear all, with the map updating live.
- **Collection browser.**

Scaffolded in Phase 5 (consult the frontend-design skill first).
